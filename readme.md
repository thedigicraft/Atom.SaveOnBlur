# Atom Save on Blur

This is a script that will allow you to easily turn a basic form field (input, textarea) into an field that will save once the user has clicked away from the field.

###This script uses:
<blockquote>
<ul>
<li>HTML</li>
<li>CSS</li>
<li>Javascript/jQuery</li>
<li>PHP</li>
<li>MySQL</li>
<li>AJAX</li>
</ul>
</blockquote>
